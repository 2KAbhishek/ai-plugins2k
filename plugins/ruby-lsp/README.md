# ruby-lsp

Ruby language server for Claude Code, providing code intelligence features like go-to-definition, find references, and error checking.

## Supported Extensions
`.rb`, `.rake`, `.gemspec`, `.ru`, `.rbx`, `.rbw`

## Installation

Install the Ruby LSP gem globally:

```bash
gem install ruby-lsp
```

Or add it to your project's Gemfile:

```ruby
gem 'ruby-lsp', group: :development
```

Then run:

```bash
bundle install
```
